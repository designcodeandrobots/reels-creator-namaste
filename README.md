# reels-analyzer — пайплайн анализа блогера и создания скила

Привет 👋 Я помогу тебе **проанализировать рилсы любого блогера** и собрать на их основе рабочий инструмент.

Маршрут такой:
1. Скачаем данные по последним **100 рилсам** блогера.
2. Скачаем аудио из этих роликов.
3. Переведём аудио в текст (транскрипция).
4. Сделаем **анализ** контента и сохраним его в файл.
5. Соберём **скил**, который умеет искать темы и писать сценарии в стиле этого блогера.
6. Дальше на этом можно строить **контент-план**.

Это репозиторий-инструкция: основной файл — [`playbook.md`](./playbook.md). Он написан так, чтобы по нему мог пройти **чат-бот / ИИ-агент по шагам**, останавливаясь на контрольных точках.

## Используемые инструменты
- **Парсер данных:** [insta-reals-parser](https://github.com/designcodeandrobots/insta-reals-parser) — выгружает таблицу по рилсам блогера (включая URL роликов).
- **Скачивание аудио:** Downie (Mac, платно) или бесплатные альтернативы — yt-dlp, JDownloader2, cobalt.tools (см. playbook, шаг 2).
- **Транскрипция:** [voice-to-text](https://github.com/designcodeandrobots/voice-to-text) — бесплатно, через локальную LLM (без облака и оплаты).
- **Анализ + скил:** ИИ-агент (этот чат) по инструкции из `playbook.md`.

## Быстрый старт
Открой `playbook.md` и отдай его чат-боту целиком (или скажи: «работай по playbook.md, шаг за шагом»). Бот проведёт тебя от установки парсера до готового скила.
